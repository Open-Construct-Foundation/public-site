CDK Patterns

When you start building serverless systems you quickly realise that your architecture is decomposed into lots of small pieces. These small pieces can be put together in multiple different ways depending on what you are trying to achieve. http://cdkpatterns.com is designed to be a collection of all the different ways you can compose these pieces into real software. By utilizing the power of aws cdk you can just clone the repo, install the dependencies, run the tests and deploy the pattern into your account to truly play/learn about what's possible. The patterns targeted for collection are the ones that the aws serverless heroes are sharing on stage at conferences meaning you have plenty of reading material to dig deeper if you want. The project is opensource too so please feel free to contribute some patterns of your own!

AWS-provided Examples

AWS also maintains CDK Examples: https://github.com/aws-samples/aws-cdk-examples as part of their Samples github repo.

More examples will come later. If you'd like to contribute, please open an Issue on the Github page here.
